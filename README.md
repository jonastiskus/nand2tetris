# Nand2tetris

Repository for nand2tetris projects.

## Purpose

- To learn and understand how computers work in hardware level by building one, starting from NAND chip.
- To learn and understand how computers work in software level by writing our own programming language and its compiler. Create virtual machine language and translator. Make operating system to run on.
- To become a better software developer :)

## Projects
 - Projects 1-3
	 - Making chips starting from NAND up to making REGISTER and RAM chips in hardware description language (HDL).
- Project 4
	- Playing with Hack assembly. Wrote number multiplication and screen fill programs.
- Project 5
	- Making ALU (Arithmetic Logic Unit) and CPU from chips that we made earlier and connect those components into single general computer.
- Project 6
	- Making our own assembler in C to turn Hack assembly language into pure bit instructions to load for our CPU.
